## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command)
I have used bcrypt 5.0.1 as 2.0.0 could not be downloaded

.
- Run the development web server using the `node express_server.js` command.
